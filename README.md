# 🎨 My Design System
Este es el repositorio central de mi sistema de diseño, donde la estética de Figma se encuentra con la eficiencia de **Tailwind CSS**.

## 🚀 Principios de Diseño
* **Consistencia:** Cada componente sigue una escala atómica.
* **Escalabilidad:** Pensado para crecer mediante Design Tokens.

## 🛠 Stack Técnico
* **Diseño:** Figma
* **Estilos:** Tailwind CSS
* **Control de versiones:** GitHub
